# 导航结构

> **版本**: v1.0.0  
> **最后更新**: 2025-01-01

## 主导航

### 导航栏结构
```
[Logo] [首页] [知识库] [产品库] [观察站] [术语] [文章] [工具] [搜索] [关注]
```

### 导航项

| 导航项 | URL | 说明 | 优先级 |
|--------|-----|------|--------|
| 首页 | `/` | 网站首页 | 高 |
| 知识库 | `/knowledge` | 技术原理知识库 | 高 |
| 产品库 | `/database` | 产品数据库 | 高 |
| 观察站 | `/news` | 新闻聚合 | 中 |
| 术语 | `/glossary` | 术语词典 | 中 |
| 文章 | `/articles` | 深度文章 | 中 |
| 工具 | `/tools` | 交互工具 | 低 |

### 导航设计
- **位置**: 顶部固定
- **样式**: 深色背景，青色高亮当前页
- **响应式**: 移动端折叠为汉堡菜单

### 响应式导航规则

| 设备 | 显示方式 | 导航项数量 |
|------|---------|-----------|
| 桌面端 (> 1024px) | 水平导航栏 | 显示全部8个导航项 |
| 平板端 (768px - 1024px) | 水平导航栏 | 显示前5个，其余折叠到"更多" |
| 移动端 (< 768px) | 汉堡菜单 | 全部折叠，点击展开侧边栏 |

**移动端折叠优先级**:
1. Logo（始终显示）
2. 搜索图标（始终显示）
3. 关注按钮（始终显示）
4. 汉堡菜单（始终显示）
5. 其他导航项（折叠到侧边栏）

## 面包屑导航

### 结构
```
首页 > 知识库 > 芯片制程 > PowerVia
```

### 规则
- 最多3级导航
- 最后一级不链接
- 使用 `>` 分隔符
- 字体：mono，颜色：secondary
