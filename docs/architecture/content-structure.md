# 内容结构

> **版本**: v1.0.0  
> **最后更新**: 2025-01-01

## 内容类型

### 1. 技术原理知识库
- **路径**: `/knowledge/[category]/[slug]`
- **分类**: 芯片制程、能效指标、架构设计
- **更新频率**: 季度评审，年度大更新
- **内容特点**: 系统化、深度、长期有效

### 2. 产品数据库
- **路径**: `/database/chips/[id]`
- **分类**: 移动芯片、桌面CPU、GPU
- **更新频率**: 每月更新
- **内容特点**: 数据驱动、实时、对比性强

### 3. 新闻聚合
- **路径**: `/news/[date]-[slug]`
- **分类**: 行业速报、深度解读
- **更新频率**: 每日更新
- **内容特点**: 时效性强、高频、短篇

### 4. 术语词典
- **路径**: `/glossary/[term]`
- **分类**: 技术术语、能效指标
- **更新频率**: 按需更新
- **内容特点**: 解释性、链接性强

### 5. 深度文章
- **路径**: `/articles/[slug]`
- **分类**: 按标签分类
- **更新频率**: 每周2-3篇
- **内容特点**: 深度分析、长文、SEO友好

## 内容分类体系

### 技术原理分类
- **芯片制程**: PowerVia、RibbonFET、FinFET等
- **能效指标**: PPW、TDP、TGP等
- **架构设计**: Apple Silicon、Blackwell等

### 产品分类
- **移动芯片**: A18 Pro、Snapdragon 8 Elite等
- **桌面CPU**: Intel 18A、AMD等
- **GPU**: RTX 5090、Blackwell等

### 新闻分类
- **行业速报**: 每日新闻
- **深度解读**: 每周深度分析

## 内容组织方式

### 标签系统
- 每个内容至少1个标签
- 标签用于聚合和推荐
- 标签页面：`/tags/[tag]`

### 相关推荐
- 基于标签匹配
- 基于内容类型
- 基于用户行为
