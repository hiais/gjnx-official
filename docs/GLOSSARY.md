# 术语表

> **版本**: v1.0.0  
> **最后更新**: 2025-01-01  
> **用途**: 统一专业术语定义，提高文档可读性

## A

### Astro
一个静态网站生成框架，特点是无 JavaScript 运行时，提供极致的性能。

### ARIA
Accessible Rich Internet Applications，用于增强网页可访问性的属性集合。

## C

### CSP
Content Security Policy，内容安全策略，用于防止 XSS 攻击。

### CI/CD
Continuous Integration / Continuous Deployment，持续集成/持续部署。

## D

### DRY
Don't Repeat Yourself，不要重复自己，代码复用原则。

## F

### Frontmatter
Markdown 文件顶部的 YAML 元数据，用于定义文章的标题、日期、标签等信息。

```yaml
---
title: "文章标题"
date: 2025-01-01
tags: ["标签1", "标签2"]
---
```

## G

### GDPR
General Data Protection Regulation，欧盟通用数据保护条例。

## H

### HSTS
HTTP Strict Transport Security，强制使用 HTTPS 的安全策略。

## J

### JSON Schema
用于验证 JSON 数据结构的规范。

## L

### Lighthouse
Google 开发的网页性能和质量分析工具。

## M

### Mermaid
用于创建图表和流程图的 Markdown 扩展语法。

## O

### OWASP
Open Web Application Security Project，开放 Web 应用程序安全项目。

## P

### Props
组件的属性，用于传递数据给组件。

## S

### Schema
数据结构定义，用于验证数据的格式和内容。

### SEO
Search Engine Optimization，搜索引擎优化。

## T

### TypeScript
JavaScript 的超集，添加了静态类型检查。

## V

### Vercel
一个云平台，用于部署静态网站和服务器less 函数。

## W

### WCAG
Web Content Accessibility Guidelines，网页内容可访问性指南。

### Web Vitals
Google 定义的核心网页性能指标（LCP、FID、CLS）。
