## 内容创建流程

### 完整工作流

```mermaid
graph TD
    A[确定内容需求] --> B{是否需要大纲?}
    B -->|重要内容| C[设计详细大纲]
    B -->|快速内容| D[选择基础模板]
    
    C --> C1[参考 outline-design-guide.md]
    C1 --> C2[设计详细大纲]
    C2 --> C3[大纲评审]
    C3 --> C4[保存大纲]
    C4 --> E[基于大纲生成内容]
    
    D --> D1[参考 templates.md]
    D1 --> D2[选择对应模板]
    D2 --> E
    
    E --> E1[填充Frontmatter]
    E1 --> E2[生成内容]
    E2 --> E3[质量检查]
    E3 --> E4[验证构建]
    E4 --> E5[自动发布]
```

### 快速参考

#### 场景1: 重要内容（推荐使用大纲）
1. 阅读 `outline-design-guide.md` 了解大纲设计方法
2. 参考 `templates.md` 选择基础模板
3. 设计详细大纲
4. 基于大纲生成内容

#### 场景2: 快速内容（直接使用模板）
1. 参考 `templates.md` 选择对应模板
2. 直接生成内容
3. 质量检查
