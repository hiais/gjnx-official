# SEO规范

> **版本**: v1.0.0  
> **最后更新**: 2025-01-01

## 元数据规范

### Title标签
- **格式**: `{标题} | 硅基能效`
- **长度**: ≤ 60字符
- **要求**: 包含主关键词
- **示例**: `Intel 18A技术分析 | 硅基能效`

### Description标签
- **长度**: ≤ 120字符
- **要求**: 吸引人、包含关键词
- **格式**: 一句话描述 + 价值点
- **示例**: `深度解析Intel 18A背面供电技术，了解芯片制程最新突破...`

### Keywords标签
- **数量**: 3-5个关键词
- **格式**: 逗号分隔
- **来源**: 文章tags + 主关键词

## 结构化数据

### Article类型
```json
{
  "@type": "Article",
  "headline": "文章标题",
  "datePublished": "2025-01-01",
  "author": {
    "@type": "Organization",
    "name": "硅基能效"
  }
}
```

### Product类型
```json
{
  "@type": "Product",
  "name": "产品名称",
  "description": "产品描述"
}
```
