## 使用建议

### 对于 AI Agent
1. **首次使用**: 先阅读 `AI_AGENT_GUIDE.md` 的"快速参考"部分
2. **执行任务**: 根据任务类型选择对应的场景（A/B/C/D）
3. **遇到问题**: 查看"紧急修复指南"或"异常处理"章节
4. **验证结果**: 运行 `npm run build` 并检查清单
