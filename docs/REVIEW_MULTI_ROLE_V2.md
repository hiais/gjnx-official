  ```markdown

  - [ ] 颜色对比度 ≥ 3:1 (UI组件)
  - [ ] 所有图片有alt属性
  - [ ] 表单有label标签
  - [ ] 键盘导航支持
  - [ ] 焦点指示器可见
  - [ ] 页面有标题 (H1)
  - [ ] 语言属性设置 (lang="zh-CN")
  ```

#### 问题2: 组件规范缺少ARIA属性说明
**位置**: `design-system/components.md`
- **问题**: 组件规范没有说明ARIA属性的使用
- **建议**: 补充ARIA属性规范：
  ```markdown
  ### ARIA属性使用
  - **按钮**: `role="button"`, `aria-label` (如需要)
  - **导航**: `role="navigation"`, `aria-label="主导航"`
  - **模态框**: `role="dialog"`, `aria-modal="true"`
  - **表单**: `aria-required`, `aria-invalid`
  ```

#### 问题3: 响应式规范缺少可访问性考虑
**位置**: `design-system/responsive.md`
