# 颜色规范

> **版本**: v1.0.0  
> **最后更新**: 2025-01-01  
> **来源**: `src/styles/global.css`

## 颜色系统

### 背景色

| Token | Hex | 用途 | 使用场景 |
|-------|-----|------|---------|
| `--c-bg-dark` | `#050505` | 主背景 | 所有页面背景 |
| `--c-bg-panel` | `#0a0a0a` | 面板背景 | 卡片、模态框 |
| `--c-bg-card` | `#111111` | 卡片背景 | 内容卡片 |

### 文本色

| Token | Hex | 用途 | 使用场景 |
|-------|-----|------|---------|
| `--c-text-primary` | `#e0e0e0` | 主文本 | 正文、标题 |
| `--c-text-secondary` | `#888888` | 次要文本 | 标签、元数据 |

### 品牌色

| Token | Hex | 用途 | 使用场景 |
|-------|-----|------|---------|
| `--c-accent-cyan` | `#00f0ff` | 数据/链接 | 可点击元素、数据点、正趋势 |
| `--c-accent-amber` | `#ffae00` | 警告/能量 | 警告信息、高能量状态、交互 |
| `--c-accent-rose` | `#ff0055` | 关键/错误 | 错误信息、高功耗状态 |

## 使用规则

### 颜色选择
- **数据展示**: 使用 `--c-accent-cyan`
- **警告信息**: 使用 `--c-accent-amber`
- **错误状态**: 使用 `--c-accent-rose`
- **链接**: 使用 `--c-accent-cyan`

### 对比度要求
- 文本与背景对比度 ≥ 4.5:1 (WCAG AA)
- 链接与背景对比度 ≥ 3:1
- 所有颜色在深色背景下可读

### 禁止使用
- 禁止添加新的品牌色（除非设计系统更新）
- 禁止使用纯白色背景
- 禁止使用低对比度颜色组合

## 扩展规则

### 新增颜色
1. 在 `global.css` 中定义CSS变量
2. 在此文档中注册
3. 更新版本号
4. 记录变更日志

### 颜色变更
1. 评估对现有组件的影响
2. 更新所有使用场景
3. 更新版本号
4. 通知开发团队
