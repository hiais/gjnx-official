# 响应式设计规范

> **版本**: v1.0.0  
> **最后更新**: 2025-01-01

## 断点定义

| 设备 | 宽度 | 断点名称 | CSS媒体查询 |
|------|------|---------|------------|
| 移动端 | < 768px | `mobile` | `@media (max-width: 767px)` |
| 平板 | 768px - 1024px | `tablet` | `@media (min-width: 768px) and (max-width: 1023px)` |
| 桌面 | > 1024px | `desktop` | `@media (min-width: 1024px)` |

### CSS实现示例

```css
/* 在 global.css 或组件样式中使用 */
.container {
  padding: 1rem;
}

/* 移动端 */
@media (max-width: 767px) {
  .container {
    padding: 0.5rem;
  }
}

/* 平板端 */
@media (min-width: 768px) and (max-width: 1023px) {
  .container {
    padding: 0.75rem;
  }
}

/* 桌面端 */
@media (min-width: 1024px) {
  .container {
    padding: 1.5rem;
  }
}
```

## 响应式规则

### 布局
- **移动端**: 单列布局，垂直堆叠
- **平板**: 2列布局（如适用）
- **桌面**: 多列布局，充分利用空间

### 导航
- **移动端**: 汉堡菜单，侧边栏导航
- **桌面**: 顶部水平导航栏

### 字体
- **移动端**: 字体大小缩小10-15%
- **行高**: 保持不变
- **字重**: 保持不变

### 组件
- **卡片**: 移动端全宽，桌面端网格布局
- **表格**: 移动端横向滚动，桌面端正常显示
- **图表**: 移动端简化，桌面端完整展示

## 移动端优化

### 触摸优化
- 点击目标 ≥ 44px × 44px
- 按钮间距 ≥ 8px
- 避免悬停效果（移动端无效）

### 性能优化
- 图片懒加载
- 减少动画效果
- 优化字体加载
