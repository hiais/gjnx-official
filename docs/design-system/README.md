## 设计原则

### 核心原则
1. **赛博朋克工程风格** - 精密、数据驱动、高对比度、仅深色模式
2. **HUD界面感** - 像高级机器的抬头显示器，而非普通博客
3. **性能优先** - 所有设计决策考虑性能影响
4. **可访问性** - 符合WCAG AA标准

### 设计关键词
- Precision（精密）
- Data-Driven（数据驱动）
- High-Contrast（高对比度）
- Dark Mode Only（仅深色模式）

## 使用指南

### 新组件开发
1. 查看 [组件规范](./components.md) 了解现有组件
2. 遵循设计原则和颜色/字体规范
3. 在组件规范中注册新组件
4. 更新版本历史

### 设计变更
1. 更新相关规范文档
2. 更新版本号
3. 记录变更日志
4. 通知开发团队
