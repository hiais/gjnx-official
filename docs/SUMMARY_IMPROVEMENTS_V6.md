### 3. 数据验证脚本 ✅

**文件**: `scripts/validate-data.js`

**功能**:
- ✅ 验证chips.json数据格式
- ✅ 检查ID唯一性
- ✅ 验证数据类型和范围
- ✅ 检查必填字段
- ✅ 输出详细的错误和警告信息
- ✅ 集成到package.json (`npm run validate:data`)

**验证规则**:
- ID格式: 小写字母+数字+连字符
- 类型枚举: mobile, desktop, gpu, laptop
- 数值范围: watts (0.1-500), score (1-10000)
- 日期格式: YYYY-MM-DD

**价值**:
- 自动化数据验证，防止错误数据进入系统
- 提供详细的错误信息，便于快速修复
- 可集成到CI/CD流程

**工作流流程**:
```
validate-data → sync-content → build → deploy
```

**价值**:
- 完全自动化的内容同步和部署流程

## 使用指南

### 1. 数据验证
```bash
# 验证数据
npm run validate:data

# 或在CI/CD中自动执行
# .github/workflows/content-sync.yml 已集成
```

### 2. 内容同步
```bash
# 手动触发内容同步
npm run sync

# 或通过GitHub Actions手动触发
# Actions → Content Sync → Run workflow
```

### 3. 使用提示词模板
参考 `docs/AI_PROMPT_TEMPLATES.md`，选择合适的模板，替换变量后使用。

### 4. 监控指标
- 查看GitHub Actions工作流状态
- 检查 `.agent/workflows/logs/` 日志文件
- 查看GitHub Issues中的告警
